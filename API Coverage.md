### Get Account by ID

```PowerShell

```

### Get Accounts

```PowerShell
Get-S1Account -Name <string> -AccountID <string[]> -Count <int> -SortBy {accountType | activeAgents | createdAt | expiration | id | name | numberOfSites | state | updatedAt} -SortOrder {asc | desc} -CountOnly
```
